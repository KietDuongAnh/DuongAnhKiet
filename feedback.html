Feedback

Feedback 1

Overall, this is a good idea that meets a current demand trend in society, which is the incorporation of AI technology into applications. It provides a practical solution for users to efficiently manage their work when faced with the problem of increasing workload. More specifically, the AI-powered automatic task classification feature will provide users with new experiences while also increasing work efficiency. However, you can think about how to use the app's reminder feature. Because many applications, such as Google Calendar, are already equipped with this feature, which has become familiar to users for a long time. Using two similar features at the same time will easily confuse users and cause many other conflicts. To fix it, you can choose to integrate this feature with the applications already on the device to provide the most convenience for users.

Reflection

First, this was valuable feedback that assisted me in identifying a new issue with my app. As a result, I gained some experience to help me improve my project. This problem not only responds to a lack of improvement, but also helps me realize a lesson for future development that I need to look at and evaluate a problem from many perspectives in order to have different solutions, and the key is to create convenience for users. In terms of improvement, integrating the application's features with those already available on the user's device is an innovative idea. However, it also raises many unforeseen problems. As you can see, everyone's needs are different and the applications they use are also diverse, so integrating them with my application is a technical challenge. Moreover, if we integrate them with an unverified application, it can easily create bugs, leak user information and other inconsistencies.
