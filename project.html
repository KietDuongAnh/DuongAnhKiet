Project Idea

Overview

My project would be about a mobile phone application using AI technology to help users get organized stay on top of their work routines. Individuals can use this task management tool to turn complex tasks into manageable to-do lists by setting daily goals and breaking them down into smaller tasks. To avoid being distracted by too many tasks, users can also set priorities for the most important ones to be completed before moving on to others. It also allows users to set and change deadlines for their goals on a flexible schedule. Also, the application's measurement would be used to keep track of these goals' progress over time, and users can view regular reports using intuitive charts. Finally, to make this application more interesting and increase use motivation, I also add an achievement-recording feature for complete goals, so that users can share them with others.

Motivation

The enhanced convenience and modernity of life means that the amount of human workload is constantly increasing. Therefore, the management of tasks is critical to the successful work. In fact, “Of the time given to a workday, 80% is spent doing tasks with little to no value and only 20% is spent doing something important” (Time Management Statistics and Facts That Will Surprise You - Trafft, 2021) [1]. As a result, we need suitable tasks management to avoid wasting time and optimize the productivity. Paper to-do lists are perfectly acceptable, but going digital has its advantages. A task can also be enhanced with the addition of notes, links, and files using these tools. This is my motivation to create this application. When building and developing this product, I always keep in mind three prerequisites: helping users get arranged, improving their working styles, and raising their focus. To-do list apps are available in a variety of forms today. While a number of these programs are widely used because of their intuitive user interfaces, others are valued for their powerful features. However, in this project, my product would strike a balance between straightforwardness and efficiency because I want it to appeal equally to novices and professionals.

Description

First of all, to make an easy access, my product would be free for users to download and enjoy. This application is supported for both Android and IOS devices. It can also make profits from the premium version, which have enhanced features such as more reports, decorations and notification via emails. Moreover, I expect my application would be a harmonious blend of simple colors and a convenience of having because it is always beneficial to have a tool that is both visually appealing and simple to use.

There are four core features that I want to focus on while developing this application:

  • Task planning and scheduling: This feature allows users to create lists of necessary goals and set deadlines for them. Furthermore, they can break their goals into smaller tasks. Users can also change, delete, or write additional comments, if necessary, for these tasks by using available edit tools of this application. It also makes use of AI technology to automatically categorize the goals, allowing users to use the application in the most convenient and effective manner. Of course, this is an optional feature, and users can disable it if they prefer a manual approach.

    • Source: Todoist

  • Task priority: My product would include a task prioritization feature to help users prevent wasting valuable resources on less important tasks. Users can mark their goals with different priority levels to avoid situations in which some minor tasks consume too much attention while relevant or really important goals go unachieved. The application will then automatically arrange them at the top of the list in descending order based on the level that the user selects. Additionally, users can set reminders to remind them of the tasks that have been labeled and are coming up to help them stay on schedule.

    • Source: nTask

  • Reporting: This is another feature that users see tangible progress of their goals. As part of task management, users would need to keep track of and observe the progress of your their activities. In terms of avoiding missing deadlines and make tasks progress more efficiently, it is necessary to control the flow of tasks. In particular, as a result of visual graphs and specific data, users would be updated on their working progress on a daily or weekly basis, see their productivity trends and stay on target to accomplish their goals. Besides, users are able to access the activity history. Specifically, this application would provide users with a task completed archive in case they want to verify if their initial plan was flawed. When users review their process, they can see how small steps can contribute up to big results.

    • Source: Todoist

  • Achievement sharing: When all of the tasks for a goal have been completed, the system will record the completed goal as a new achievement and save it to the user's account, which they can share with other users. This feature can be used for a variety of purposes. Members of a project team, for example, can share achievements to track their progress, or users can simply show off their accomplishments to friends and motivate them to work harder toward their goals

Tools and technologies

Software

  • Native mobile development tools: Native development tool is software that enables developers to create applications for a single platform. To begin, Xcode is an integrated development environment (IDE) that is used to create software for macOS or iOS. It provides source code for the most widely used iOS programming languages, including Swift and Objective-C, as well as a variety of programming models. In addition, Android Studio (Android) is another ideal tool for mobile application builder. It supports programming languages for Android development such as Java and Kotlin.
  • Cross-platform development tools: React Native is an cross-platform tools that can be used to create both Android and iOS software. In addition, its flexibility is enhanced by the fact that it can interpret code and convert it into native ones in a short time. Besides, OpenCV (Open Source Computer Vision Library) is another necessary tool for AI function in this application. It is a collection of programming functions for real-time computer vision and machine learning. It has C++, Python, and Java interfaces and supports both iOS and Android.

I chose these tools for my project because they are both free to use and credible due to their widespread use by high-tech companies around the world.

Hardware

A macOS device or Windows device for application developing and mobile phones with latest iOS and Android versions for testing purpose.

Outcome

If my project is successful in the future, it will make a significant contribution to society. In particular, because everyone's lifestyle will improve in a positive direction, there is no longer a need to worry about managing and distributing work. Specifically, people will become more organized and stay focused with the help of a task management system. Moreover, the productivity of organizations or, more broadly, the industry related to human labor will be improved if each of their individuals knows how to work effectively. As a result, they can provide more qualified products, which is an advantage in the competitive market.
